* Void linux
** Installation
1. Install via the ISO
2. Update voidlinux
3. Install git and ansible
4. Run the installation
   ~sudo ansible-pull -U https://gitlab.com/venikx/personal-ops.git void-linux/setup.yml~
5. Well played, mate.
